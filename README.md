## Gesture-recogniser-Ubuntu-12.04

Gesture recognition - Tracks a yellow object and recognises the pattern drawn (Tested on Ubuntu-12.04 only)

#### Required :
1) Front Camera
2) Ubuntu-12.04 Operating System

#### Required Python modules:
* numpy
* cv2
* sys
* Image
* ImageDraw
* colorsys
* os
* time
* math
* kNN
* easygui
* operator
